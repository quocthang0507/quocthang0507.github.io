Information
  • MD5: 128-bit hash, fast but not recommended for security (vulnerable to collisions)
  • SHA-1: 160-bit hash, deprecated for security but still used for checksums
  • SHA-256: 256-bit hash, widely used and secure for most applications
  • SHA-384: 384-bit hash, more secure than SHA-256, good for sensitive data
  • SHA-512: 512-bit hash, highest security level, used for critical applications
  • Privacy: All hashing is performed locally in your browser. No data is sent to any server.
